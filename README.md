### Step 3 — Generate Your Auth Key & API Key

1. In Tailscale (browser), go to **Settings → General → Keys**
2. Generate an **Auth Key** — make sure to turn on **Reusable**, then save it somewhere safe

<img width="697" height="878" alt="Auth key settings" src="https://github.com/user-attachments/assets/fd0e2a01-684c-4b1b-a6e2-131ae1907194" />

<img width="663" height="490" alt="Auth key created" src="https://github.com/user-attachments/assets/30ce9869-3820-4664-a6c8-1fcdfcf72ebf" />

3. Generate an **API Key** and save it somewhere safe as well

<img width="656" height="425" alt="API key created" src="https://github.com/user-attachments/assets/c5d910c4-0991-4692-ba79-127d9d88a0c9" />

### Step 4 — Run the Workflow

1. Go to the **Actions** tab in your forked repo and tap **"I understand my workflows"**
2. Under **All Workflows**, tap **"Click Here To Run"**

<img width="810" height="1440" alt="Actions tab" src="https://github.com/user-attachments/assets/230358f9-0206-4443-b405-419200c2a8b8" />

<img width="809" height="1440" alt="Run workflow" src="https://github.com/user-attachments/assets/2a8378b2-0ddf-4105-91d2-fe6048ef3a64" />

3. Enter your **Tailnet**, **Auth Key**, and **API Key** when prompted, then run it

<img width="809" height="1440" alt="Workflow inputs" src="https://github.com/user-attachments/assets/386889c9-9405-4dea-ac43-a32b481f6e27" />

4. Click on **rdp** and wait until it reaches the **Keep Alive** step — if it's spinning there, you're good to go ✅

<img width="809" height="1440" alt="Keep alive step" src="https://github.com/user-attachments/assets/b0a54227-fc7d-42ab-9dc5-0cd35a1c8542" />

---

### Step 5 — Connect via Your Phone

1. Open **Windows App** (or **Windows App Mobile** on iOS), tap **Accept → Skip → PC Connection**
2. Tap **Add PC** and enter the following:

| Field | Value |
|-------|-------|
| PC Name | `gubby` |
| Username | `GubbySamurai` |
| Password | `Gubby@12345` |

<img width="811" height="1440" alt="Add PC screen" src="https://github.com/user-attachments/assets/fe2789c3-081b-40a4-81d5-15aa19958e2e" />

<img width="809" height="1440" alt="User account screen" src="https://github.com/user-attachments/assets/ed0ebae3-1acc-47c8-896b-da832fb2279b" />

3. Tap **Add**, then **Connect**
4. Once you're in, run the **Opera GX Setup** on the desktop, install it, and open the browser to claim your reward! 🎉
